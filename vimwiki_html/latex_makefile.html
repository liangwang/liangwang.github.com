LaTeX-Makefile

Chris Monson has developed an amazing makefile (GoogleCodes) for LaTeX. It has many features such as generating graphics from data, colorized (therefore, pretty) error output, and so on (more). I slightly modified it to suit my own needs. My version can be checked out from bitbucket or github

This makefile is so versatile that it almost suits all situations I have workes with LaTeX document. Instead of copy this makefile into each individual latex projects, I made a BASH alias to refer to one single makefile: 

alias lmake="make --file=/PATH/TO/LATEX-MAKEFILE"
